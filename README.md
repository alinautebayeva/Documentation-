# Примеры документов

Данный репозиторий содержит комплект документации 
для приложения бронирования переговорных комнат внутри компании (MeetRoom).

Репозиторий создан в качестве портфолио и включает в себя:
- Software Requirement specification для продукта MeetRoom: [SRS](https://github.com/alinautebayeva/Documentation-/blob/main/SRS.md);
- Описание методов REST API: [API Specification](https://github.com/alinautebayeva/Documentation-/blob/main/API%20Specification.md);
- UML-диаграмму последовательности для процесса бронирования переговорной комнаты: [Sequence_diagram](https://github.com/alinautebayeva/Documentation-/blob/main/Sequence_diagram.pdf)
